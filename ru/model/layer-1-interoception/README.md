## Связь с другими слоями

Слой 0 (Автономный профиль) описывает, *в каком режиме* находится твоя нервная система. Слой 1 спрашивает: *а ты вообще это чувствуешь?*

Интероцепция — это мост между состоянием тела и сознательным опытом. Без неё автономные режимы остаются невидимыми — ты можешь быть глубоко в мобилизации и не осознавать этого, пока не сорвёшься на кого-то. С хорошей интероцепцией ты ловишь сигнал раньше и имеешь больше пространства для осознанной реакции.

Слой 1 связан и с тем, что будет дальше: интероцептивный профиль влияет на то, как ты переживаешь эмоции (Слой 2+), потому что, согласно современным исследованиям, эмоции отчасти конструируются из интероцептивных сигналов.
