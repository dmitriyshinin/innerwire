## Связь с другими слоями

Паттерн привязанности и автономная регуляция глубоко переплетены. Твои самые ранние отношения с теми, кто о тебе заботился, буквально сформировали настройки по умолчанию для твоей нервной системы (Слой 0). Ребёнок, чей взрослый был стабильно доступен, вырос с нервной системой, которая ожидает безопасности в близости. Ребёнок, чей взрослый был непредсказуем или недоступен, вырос с системой, которая воспринимает близость как потенциальную угрозу.

Это значит, что паттерн привязанности — это не только «психология», у него есть физиологическая основа. Когда ты тревожишься из-за того, что партнёр не отвечает на сообщение, — это твоя автономная нервная система переключается в режим мобилизации. Когда ты эмоционально отключаешься после конфликта — это может быть активация дорсального вагуса (режим замирания).

Работа с привязанностью и работа с автономной регуляцией усиливают друг друга. Расширяя свою способность выдерживать близость (Слой 3), ты одновременно расширяешь доступ к режиму связи (Слой 0). И наоборот.
