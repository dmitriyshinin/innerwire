# InnerWire — Открытая научно обоснованная система самопознания

*Inner — внутренний. Wire — проводка. Твоя внутренняя проводка — нервная система, восприятие, паттерны реагирования.*

## Что это

InnerWire — это модель того, как ты функционируешь: нервная система, восприятие, реакция на стресс, паттерны отношений. Профиль по нескольким осям, каждая из которых опирается на рецензированные исследования. Не ярлык, с которым нужно смириться, а карта, по которой можно двигаться. Понять, где ты сейчас, что на это влияет, и что можно изменить.

## Для кого

- Для тех, кто хочет разобраться в себе — и сделать что-то с этим пониманием
- Для тех, кто ценит доказательную базу и хочет видеть ссылки на источники
- Для тех, кто ищет не только описание, но и конкретные практики
- Для специалистов, которым нужна интегративная рамка с опорой на исследования

## Принципы

1. **Научная основа** — каждое утверждение подкреплено ссылками на рецензированные исследования
2. **Слои сложности** — каждый слой самодостаточен и полезен сам по себе
3. **Спектры, а не типы** — люди различаются по континуумам, а не по дискретным категориям
4. **Динамика** — профиль не приговор, он меняется в зависимости от состояния, контекста, практики
5. **Открытость** — модель открыта для критики, дополнения и развития

## Дисклеймер

InnerWire — это инструмент самопознания и саморазвития, а не медицинский диагностический инструмент. Система не заменяет консультацию специалиста. Если вы переживаете серьёзные трудности с психическим здоровьем, пожалуйста, обратитесь к квалифицированному психологу или психотерапевту.
