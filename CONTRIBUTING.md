# Contributing / Как участвовать

## Principles / Принципы

InnerWire is an open project. We welcome contributions, but ask that you follow these principles:

### 1. Scientific basis / Научная обоснованность
Every claim must be backed by a reference to a peer-reviewed publication. Opinions, personal experience, and popular books are not sufficient grounds for inclusion (though they may prompt discussion).

### 2. Honesty about evidence level / Честность относительно уровня доказательности
If a study is preliminary — say so. If results are contradictory — show both sides. If something is a hypothesis — call it a hypothesis.

### 3. Accessible language / Доступность языка
Texts for the "main" level should be understandable without specialized education. Technical details go in collapsible sections (details/summary) or separate files.

### 4. Bilingual / Двуязычность
Content should be available in both Russian and English. If you can only write in one language — that's fine, submit what you have, others can translate.

## What you can do / Что можно делать

- **Add sources** — found a relevant study? Suggest it via PR
- **Fix errors** — factual, grammatical, structural
- **Improve descriptions** — make them clearer, more precise, more accessible
- **Suggest practices** — with mandatory scientific justification
- **Translate** — between Russian and English, or to other languages
- **Build tools** — questionnaires, trackers, visualizations
- **Critique** — if you see a weak spot, open an issue

## PR format

1. One change — one PR
2. In the PR description: what changed and why
3. New claims → attach source reference
4. Changed existing claims → explain why the new wording is more accurate
