## What is InnerWire? / Что такое InnerWire?

InnerWire is an open model of how you function — your nervous system, perception, stress response, and relationship patterns. A profile across several axes, each grounded in peer-reviewed research. Not a label to live with, but a map to navigate by.

InnerWire — открытая модель того, как ты функционируешь: нервная система, восприятие, реакция на стресс, паттерны отношений. Профиль по нескольким осям, каждая из которых опирается на рецензированные исследования. Не ярлык, с которым нужно смириться, а карта, по которой можно двигаться.

## Principles / Принципы

1. 🔬 **Scientific basis** — every claim is backed by peer-reviewed research / **Научная основа** — каждое утверждение подкреплено ссылками на исследования
2. 📊 **Spectrums, not types** — people vary along continuums / **Спектры, а не типы** — люди различаются по континуумам
3. 🧅 **Layers of complexity** — each layer is self-sufficient and useful on its own / **Слои сложности** — каждый слой самодостаточен и полезен сам по себе
4. 🔄 **Dynamic** — your profile changes with context, state, and practice / **Динамика** — профиль меняется в зависимости от контекста и практики
5. 🔓 **Open** — the model is open for critique, contribution, and evolution / **Открытость** — модель открыта для критики и развития
